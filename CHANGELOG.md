# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2024-01-28

### Added
- Initial release of PCM Interactive Book
- Complete project structure with Next.js 16
- Part I: Introduction chapters
- Part II: Origins of Modern Mathematics chapters
- Part III: Mathematical Concepts (sample chapters)
- Part IV: Branches of Mathematics (sample chapters)
- Interactive visualization components:
  - CommutativePropertyDemo
  - FunctionVisualizer
  - QuantifierDemo
  - NumberSystemsTimeline
  - EulersFormulaDemo
  - PiVisualization
  - PythagoreanTheoremDemo
- Exercise system with progress tracking
- Achievement system
- Responsive design
- Dark theme UI
- Manim animation scripts
- Deployment configuration for Netlify

### Features
- 3Blue1Brown-style visualizations
- Feynman-style explanations
- Interactive exercises with immediate feedback
- Progress tracking and achievements
- Mobile-responsive design
- Mathematical notation with KaTeX
